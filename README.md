# OmniCommerce

OmniCommerce is a commercial full-stack single-store e-commerce platform built with Next.js, TypeScript, and MongoDB.

This project was created as a personal portfolio piece to showcase full-stack architecture, admin system design, commerce workflows, modular backend structure, and product-oriented web application development.

## Features

- Customer-facing storefront
- Product listing and product detail pages
- Wishlist, cart, and checkout flows
- Customer account, orders, and address management
- Admin dashboard for catalog, inventory, orders, users, and settings
- Product variants, specifications, SEO, bundles, and relations
- Promotions, gift cards, refunds, returns, and shipment workflows
- CMS management for pages, banners, and navigation
- Authentication and role-based access control
- Modular services and repository-based backend structure
- MongoDB-backed commerce domain models

## Tech Stack

- Next.js
- React
- TypeScript
- MongoDB
- Tailwind CSS
